# 📈 סורק מניות ומדדים — מדריך הקמה מלא (לא צריך רקע טכני)

אפליקציה שמדרגת מניות, מדדים, קריפטו וסקטורים לפי ניקוד טכני, מראה מתי לקנות ומתי
למכור, כוללת בק-טסט, מעקב תיק, מחשבון סיכון והתראות טלגרם אוטומטיות.

> ⚠️ **חשוב:** זה כלי עזר לניתוח, **לא ייעוץ השקעות**. הנתונים מושהים ~15 דקות.
> אינדיקטורים לא מנבאים את העתיד. השתמש תמיד בסטופ-לוס. האחריות עליך בלבד.

## אפשרות א' — האתר בקישור (מומלץ) 🌐
כך תקבל **כתובת אינטרנט קבועה** שנפתחת מכל מחשב או טלפון, בלי להתקין כלום.

### שלב 1: חשבון GitHub (חינם)
1. היכנס ל־https://github.com והירשם (Sign up). זה חינמי לגמרי.

### שלב 2: העלאת הקוד ל-GitHub (בלי שורת פקודה!)
1. לחץ למעלה מימין על **+** → **New repository**.
2. תן שם, למשל `stock-app`, בחר **Public**, לחץ **Create repository**.
3. במסך הבא לחץ על הקישור **"uploading an existing file"**.
4. פתח את התיקייה `stock-app` במחשב שלך (בתוך Downloads), **בחר את כל הקבצים**
   (`app.py`, `engine.py`, `config.py`, `data.py`, `portfolio.py`, `requirements.txt`,
   `alerts_bot.py`, `watchlist.json`, `README_he.md`) **וגם את התיקייה `.github`**,
   וגרור אותם לחלון הדפדפן.
   > אם קשה לגרור את `.github` — אל דאגה, היא נחוצה רק להתראות (אפשר להוסיף בהמשך).
5. למטה לחץ **Commit changes**.

### שלב 3: פרסום האתר ב-Streamlit (חינם)
1. היכנס ל־https://share.streamlit.io ולחץ **Continue with GitHub** (אשר גישה).
2. לחץ **Create app** → **Deploy a public app from GitHub**.
3. בחר את ה-repo שלך (`stock-app`), ובשדה **Main file path** כתוב: `app.py`.
4. לחץ **Deploy**. בפעם הראשונה זה לוקח 1-3 דקות (מתקין ספריות).
5. 🎉 קיבלת **קישור קבוע** (משהו כמו `https://xxxx.streamlit.app`). שמור אותו,
   פתח מהטלפון, הוסף למסך הבית.

> האתר "נרדם" אחרי חוסר שימוש וקם תוך כמה שניות כשנכנסים — נורמלי בגרסה החינמית.

## אפשרות ב' — הרצה על המחשב שלך 💻
למי שרוצה לבדוק לוקאלית לפני העלאה.

1. התקן Python מ־https://www.python.org/downloads (סמן **Add to PATH** בהתקנה).
2. פתח **PowerShell** בתיקייה `stock-app` והרץ:
   ```powershell
   pip install -r requirements.txt
   streamlit run app.py
   ```
3. הדפדפן ייפתח אוטומטית בכתובת `http://localhost:8501`.

---

## שלב 4 (אופציונלי): התראות טלגרם אוטומטיות 🔔
כדי לקבל הודעות **גם כשהאתר סגור** — מנוע נפרד רץ ברקע ב-GitHub.

1. **בוט:** בטלגרם פתח שיחה עם **@BotFather**, שלח `/newbot`, עקוב אחר ההוראות,
   וקבל **TOKEN** (מחרוזת ארוכה).
2. **Chat ID:** פתח שיחה עם **@userinfobot** — הוא ישלח לך את המספר שלך (Id).
3. ב-GitHub, בתוך ה-repo: **Settings → Secrets and variables → Actions →
   New repository secret**. הוסף שני secrets:
   - שם `TELEGRAM_BOT_TOKEN`, ערך = ה-TOKEN מ-BotFather.
   - שם `TELEGRAM_CHAT_ID`, ערך = המספר מ-userinfobot.
4. לשונית **Actions** → בחר את "התראות מניות" → **Run workflow** (בדיקה מיידית).
   מעכשיו זה ירוץ אוטומטית לפי התזמון.

**לשנות אילו מניות במעקב:** ערוך את `watchlist.json` ב-repo (עריכה ישירה בדפדפן:
פתח את הקובץ ב-GitHub → אייקון העיפרון → שנה → Commit).
